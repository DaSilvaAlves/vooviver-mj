## Story

**As a** developer a implementar a landing page VooViver,
**I want** ter o projecto Astro 5 inicializado com Tailwind CSS 4, estrutura de ficheiros correcta e pipeline de deploy no Vercel configurado,
**so that** todos os stories seguintes possam começar a implementar componentes sem fricção de setup.

## Acceptance Criteria

1. Projecto Astro 5 inicializado em `vooviver-mj/` com `npm create astro@latest` — integração TypeScript estrita activada.
2. Tailwind CSS 4 instalado e funcional (`@tailwindcss/vite` como plugin Vite).
3. Estrutura de ficheiros conforme PRD Secção 7 existe: `src/layouts/`, `src/pages/`, `src/components/`, `src/data/`, `src/styles/`, `src/assets/`, `public/`.
4. `src/styles/global.css` inclui `@import "tailwindcss"` e importa as fontes Inter e JetBrains Mono via Google Fonts ou Fontsource.
5. `src/layouts/Layout.astro` criado com: `<html lang="pt-PT">`, meta charset, meta viewport, slot para conteúdo, ligação ao `global.css`.
6. `src/pages/index.astro` criado — importa `Layout.astro`, renderiza um `<main>` vazio (placeholder para componentes futuros).
7. Ficheiros de dados JSON criados com estrutura válida (podem estar vazios/com dados placeholder): `src/data/destinations.json`, `src/data/testimonials.json`, `src/data/faq.json`.
8. `public/robots.txt` criado com conteúdo básico (`User-agent: *`, `Allow: /`).
9. `astro.config.mjs` configurado com integração Tailwind e `output: 'static'`.
10. `npm run build` executa sem erros e gera output em `dist/`.
11. Projecto configurado no Vercel (via `vercel.json` ou dashboard) — deploy automático a partir do branch `main`.
12. URL de preview Vercel funcional e acessível.

### Tailwind CSS 4 — Diferenças Críticas vs v3

Tailwind 4 usa CSS-first configuration. **NÃO existe `tailwind.config.js/mjs`** por defeito.
- A configuração é feita em CSS via `@theme` no `global.css`
- O plugin é `@tailwindcss/vite` (não `@tailwindcss/astro`)
- Import no CSS: `@import "tailwindcss"` (não `@tailwind base; @tailwind components; @tailwind utilities`)

```css
/* src/styles/global.css */
@import "tailwindcss";

@theme {
  --color-primary: #1B4965;
  --color-secondary: #E07A5F;
  --color-accent: #D4A574;
  --color-bg: #FAFAF8;
  --color-text: #2D3436;
  --color-text-secondary: #636E72;

  --font-heading: 'Inter', sans-serif;
  --font-body: 'Inter', sans-serif;
  --font-mono: 'JetBrains Mono', monospace;
}
```
